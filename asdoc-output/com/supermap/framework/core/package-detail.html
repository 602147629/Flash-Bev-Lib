com.supermap.framework.core Interfaces  | Classes
 

Flex Bev框架核心包. 主要包含了按照各个功能模块划分的接口定义,涉及到插件接口,状态接口,模板接口等以及相关接口的对应实现. 该包对了解整个框架的功能组成有一定帮助.


Interfaces
 InterfaceDescription
 IGear 基础模块接口.
 IPlugin 插件寄存在baseContianer里 是对baseComponent(插件实例)的操作 最终都是以类似操作Element的方式来进行装卸的 因为这里暂不开放PluginEvent事件 相关功能可以通过SDK内部的ElementADD等事件来进行监听即可.
 ITemplate 统一模板接口.
Classes
 ClassDescription
 BaseLayout 布局设置参数类 该类不可实例化
 GearStates 基础模块状态类.这里默认有三种状态.
