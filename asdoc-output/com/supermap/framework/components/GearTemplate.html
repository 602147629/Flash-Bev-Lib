Classpublic class GearTemplate
InheritanceGearTemplate spark.components.SkinnableContainer
Implements ITemplate

基础模板类. 该类是基础模块统一外观的逻辑封装类.继承自SkinnableContainer. 该类也可以单独定义皮肤外观文件,以便于更好的呈现统一模板.

Default MXML PropertymxmlContentFactory
