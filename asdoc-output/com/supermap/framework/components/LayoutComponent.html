Packagecom.supermap.framework.components
Classpublic class LayoutComponent
InheritanceLayoutComponent BaseComponent spark.components.supportClasses.SkinnableComponent
Implements mx.core.IDeferredContentOwner, mx.core.IVisualElementContainer
Subclasses GearContainer, LayoutContainer, TemplateContainer

布局基础组件. 该组件主要基于BaseComponent基础组件增加了布局管理的功能,布局这方面使用LayoutManager进行管理. 默认使用内置的布局管理器.

Default MXML PropertymxmlContentFactory
