com.supermap.framework.componentsClasses
 

基础组件包,主要包含了基础组件类,基础组件委托类以及委托类管理类等. 同时,该包还提供了一个布局的基础组件. 如果要定制可视化组件,可以选择继承自基础组件(BaseComponent),也可以选择继承自布局组件(LayoutComponent).


Classes
 ClassDescription
 BaseComponent 基础组件类 该类主要用来作为框架本身可视化组件的基类,同时也是MapContianer,GearContainer与TemplateContainer容器的父类.
 BaseGear 基础模块实现类.
 GearTemplate 基础模板类.
 GearTemplateSkin基础模块对应的皮肤文件,可自行写皮肤来统一修改模块的外观.
 LayoutComponent 布局基础组件.
 MapControl 框架本身用到的map类，该类继承SuperMap iClient for Flex里的map类 这里不影响对map的使用，只是扩展了map，使得该地图组件在框架里当做一个插件使用。
 SkinnableContainerSkin基础布局容器对应的皮肤文件.
