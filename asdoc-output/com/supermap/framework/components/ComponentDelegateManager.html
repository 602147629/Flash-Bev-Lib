Classpublic class ComponentDelegateManager
InheritanceComponentDelegateManager Object

组件委托管理类. 该类主要使用本身的一个单例来管理组件委托对象(ComponentDelegate). 通过该类可以检测加载到舞台上的对象是否已经被注册,也就是说只有注册到该类的对象才能正常使用框架本身的注入与命令等功能. TODO:该类开放的接口需要重新考虑一下......
